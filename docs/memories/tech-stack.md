## 通用编程原则

1. 严格禁止使用 Emoji
2. 注重模块化（多文件）和禁止单体巨文件（monolith），代码行数 > 1000；
3. 任何代码必须包含单元测试，且测试覆盖率尽量达到 100%；
4. 任何大段的代码必须包含文档注释，且注释内容须易于理解、准确，最好使用简体中文；


## Language Perference

1. Python first
2. Shell
